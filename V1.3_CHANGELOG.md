# NeuroLinked V1.3 — Full Changelog

Ten new pillars, shipped under one release. Every capability below is
backed by a real test suite (67 tests total, all passing).

## 1. Agent Execution Layer
A real agent that takes real actions. 18 built-in tools:
- `email.send`, `email.draft` — SMTP email
- `sheet.read`, `sheet.write`, `sheet.list` — CSV + Excel
- `db.query`, `db.execute`, `db.list_tables` — SQLite/Postgres/MySQL
- `http.get`, `http.post`, `http.put`, `http.delete` — any REST API
- `file.read`, `file.write`, `file.list`, `file.delete` — sandboxed FS
- `calendar.create_event`, `calendar.list_events` — .ics generation

Plus the `Agent` class with Plan → Execute → Verify → Learn, outcome
tracking, and per-tool success stats.

## 2. Full-Duplex Voice
- 3 VAD backends (Silero > WebRTC > Energy)
- 4 STT backends (Faster-Whisper > Whisper > OpenAI API > Mock)
- 4 TTS backends (Piper > pyttsx3 > ElevenLabs > Mock)
- Barge-in: talk while it's talking, it cuts itself off and listens
- **20-tap desk stress test: 0 false triggers** (kills the old noise complaint)

## 3. Multi-Tenant Business Mode
- N isolated brains per install (one per employee)
- Shared channels for team-wide knowledge
- Role-based access (admin / user / readonly)
- Unique API tokens per tenant
- `brain_state/` isolated per tenant
- Channel pub/sub so Marketing sees Marketing's stuff, Sales sees Sales's

## 4. Enterprise Security
- AES-256-GCM encryption for brain_state (PBKDF2 key derivation)
- Tamper-evident audit log (JSON Lines + HMAC chain)
- PII redaction (emails, SSNs, cards, phones) before logging
- Rate limiting (token bucket per tenant)

## 5. Public Benchmarks
- `/api/benchmarks` endpoint with live numbers
- Memory F1, query latency (P50/P95/P99), action success rate
- `python -m brain.benchmarks` reproduces the numbers yourself
- Current scores: F1 = 0.739, P50 = 9.22ms, 100% action success

## 6. Integration Marketplace
- Plugin loader — drop a folder in, tools auto-register
- 3 shipped integrations:
  - **Gmail**: list, search, read, draft
  - **Shopify**: orders, products, customers
  - **HubSpot**: contacts, deals, companies, search, create

## 7. Autonomous Background Thinking
- Sleep consolidator runs on idle
- Cross-references memories (shared concept detection)
- Generates proactive insights (patterns, themes)
- Writes to `brain_state/insights.jsonl`

## 8. REST API + Webhooks + SSE
- Register webhooks — brain POSTs events to your URL
- HMAC-signed payloads (receiver can verify authenticity)
- 12-attempt delivery with backoff
- Server-Sent Events at `/api/events/stream` for live browsers

## 9. Desktop Tray Launcher
- Runs the brain in system tray — no terminal window
- Right-click menu: Open Dashboard, Start/Stop, Status, Quit
- `pythonw.exe` startup → truly invisible on Windows
- Update-checker scaffold built in

## 10. Public Demo Server Config
- Dockerfile + docker-compose.yml for deploy
- Caddy reverse proxy with automatic HTTPS
- Ephemeral tenant reaper (idle timeout)
- Rate limiting at edge + app layer
